Webサイトを構成する技術

HTML

HTML(HyperText Markup Language)はWebページのコンテンツを組み立てるためのマークアップ言語です。コンテンツとは段落や箇条書き、画像、表などの組み合わせで構成されています。

CSS

CSS(Cascading Style Sheet)はWebページのスタイルを設定するための言語です。スタイルとは文字の色や種類、コンテンツの配置(レイアウト)などのことです。

JavaScript

JavaScriptはWebページでよく使用されるスクリプト言語です。他のプログラミング言語と同様に、変数や配列、基本的な制御構文(順次処理、分岐処理、反復処理)などが利用できる他、HTMLの要素を操作すること(DOM)や、Webブラウザー上でのユーザーの操作(イベント)を起点にプログラムを動かすことができます。

よくある質問

「正しいHTML」とはなんですか？

「構文的に正しいHTML」と「意味的に正しいHTML」のことです。

構文的に正しいHTMLとは、HTMLで定められた仕様に沿って記述されたHTMLのことです。

一方、意味的に正しいHTMLとは、マークアップした内容に対して使用している要素が適切かどうかなどで判断されます。

CSSを編集してもブラウザに反映されません。

ブラウザのキャッシュの影響が考えられます。

編集前の古いCSSのデータがブラウザのキャッシュに保存されており、その情報が読み込まれている可能性があります。

ブラウザを更新する際にキャッシュをクリアするには「ctrl+F5」でリロードしてみましょう。

JavaScripの「オブジェクト」とはなんですか？

オブジェクトとは、関連するデータや機能をひとまとまりにしたものです。

オブジェクトは一般的にプロパティ(≒変数)とメソッド(≒関数)で構成されています。
